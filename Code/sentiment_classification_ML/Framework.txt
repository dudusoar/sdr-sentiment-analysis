# 关键功能说明
1. 二分类框架:
   - OVO策略: 一对一分类
   - OVR策略: 一对多分类
   - 支持SVM、朴素贝叶斯、随机森林等模型

2. 特征工程:
   - TF-IDF: 1-3gram特征
   - Word2Vec: 300维向量(已修复)
   - 智能降级机制确保系统稳定性

3. 评估体系:
   - ROC-AUC、准确率、F1分数
   - 混淆矩阵分析
   - 性能对比报告

4. 数据处理:
   - 4,971条YouTube评论
   - 三分类标签(消极/积极/中性)
   - 完整的预处理流程
